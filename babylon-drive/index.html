Strážce Pokladu

Připoj Google Drive, aby se tvůj poklad
ukládal bezpečně do cloudu.

1 Otevři Google Cloud Console
Jdi na console.cloud.google.com a přihlas se svým Google účtem.
2 Vytvoř nový projekt
Klikni na „Vybrat projekt" → „Nový projekt". Název: strazce-pokladu → Vytvořit.
3 Povol Google Drive API
V menu vlevo: API a služby → Knihovna. Vyhledej Google Drive API → Klikni → Povolit.
4 Vytvořit OAuth přihlašovací údaje
API a služby → Přihlašovací údaje → Vytvořit → OAuth ID klienta.
Typ aplikace: Webová aplikace.
Autorizované JS origins: přidej adresu, kde běží appka (nebo http://localhost pro lokální test).
Klikni Vytvořit – zkopíruj ID klienta.
5 OAuth souhlas (pokud vyzván)
API a služby → OAuth souhlas. Typ: Externý, vyplň název apky. Obory: přidej drive.file. Do testovacích uživatelů přidej svůj Gmail.
💡 Bezpečnost: Appka si vyžádá přístup pouze ke svým vlastním souborům (ne k celému Drive). Tvá data nikam neodesíláme – vše běží přímo mezi telefonem a Google.
